# taipei-day-trip
## Purpose
基本的旅遊購物網站，<br/>
前端使用基本的原生h5、css及js建設，<br/>
後段框架使用python的flask<br/>
金流串接tappay<br/>
部署在AWS的ec2上（時間久了可能下架）
